# 🏠 PGNest: Online Paying Guest Connectivity System

PGNest is a direct, peer-to-peer web application designed to eliminate massive brokerage fees for migrating students. It connects students directly with verified property owners to ensure zero brokerage, transparent pricing, and safe stays.

## 🚀 Key Features
* **Zero Brokerage "Nest Pass":** A custom digital credit system where students use 1 credit to unlock a verified landlord's contact.
* **Automated 3-Strike Moderation:** Bad actors are automatically hidden from the search engine if they receive 3 verified user reports.
* **Role-Based Access Control:** Distinct, secure dashboards for Students, Landlords, and Admins.
* **Real-Time UI:** Asynchronous contact unlocking and chat via AJAX.

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3, Vanilla JavaScript, AJAX
* **Backend:** PHP 8.x (PDO Prepared Statements, BCrypt Password Hashing)
* **Database:** MySQL (3NF Normalized Relational Database)
* **Architecture:** 3-Tier Web Architecture
